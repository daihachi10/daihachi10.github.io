操作説明

Use the left and right arrow keys to move the bar.
Pick up the ball and earn points.


If it hits at 15%, the ball (purple) is -5,
7.5% will result in a +5 ball (yellow).

Press the left and right arrow keys while holding down the W key to move it faster.

(When I made it the control key, the browser came back and it became the W key.)

If the ball touches the red floor, the game is over.

If the game is over, you can restart with the Enter key.

Press the up arrow key to drop the ball directly below where it is now.

Press the down arrow key to make the ball fall faster.


The speed at which the ball falls increases with each 5 increase in score up to a score of 40.
After that, the rate of fall gradually increases until a constant speed is reached.

Pressing the I key adds animation.
